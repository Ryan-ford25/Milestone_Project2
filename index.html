Rules

The rules for this game are simple.

  • To start, this message will be displayed below the grid: `Computers turn`.
  • That simply means sit tight while the computer displays the sequence.
  • The computer will flash a random square so watch carefully!
  • This message will then be displayed below the grid: `Your turn`.
  • When that happens click on the same square that was lit up.
  • Nail the patter and level up - each level adds an extra square to remember!
  • You get 3 Lives. Mess up the pattern and you lose a life.
  • If your lives reach 0 its Game Over!.
  • At the end you can enter your name for the leaderboard.
  • Will you make the Top 10?
  • Good Luck, player. The grid awaits!
